- 개발기간 : 4주

## 구현기술

- 언어 : HTML, CSS, JAVASCRIPT, PHP
- 웹서버 : Apache
- 데이터베이스 : MySQL
- 운영체제 : Ubuntu
- 프로토콜 : HTTPS, SFTP, SSH, SMTP, POP3
- API/라이브러리 : jQuery, PHPMailer, Bootstrap 4, Fetch API
